Advanced Layout Features

Conditional Sections

Layouts can include conditional sections that appear based on data passed from controllers. 

<div th:if="showHero">
    <!-- Hero content -->
</div>

Multiple Content Sections

Layouts can define multiple yield points for different types of content. 

<div th:yield="sidebar"></div>
<div th:yield="content"></div>
<div th:yield="scripts"></div>
