TreeHouse CLI Commands

Comprehensive guide to all TreeHouse command-line interface tools for project management, development, and deployment.

Context-Aware CLI

Automatically detects project context and shows relevant commands

Works from Any Directory

Run commands from any subdirectory within your project

Command Grouping

Organized into logical command groups for easy discovery

Basic Usage

General Syntax

$ treehouse <command> [options] [arguments]

Get Help

$ treehouse <command> --help

Check Version

$ treehouse --version

Global Options

-h, --help Display help information

Quick Reference

Most Common Commands

treehouse serve Start development server
treehouse cache:clear Clear application cache
treehouse migrate:run Run database migrations
treehouse user:create Create new user

Command Groups

treehouse cache Show cache commands
treehouse user Show user commands
treehouse cron Show cron commands
