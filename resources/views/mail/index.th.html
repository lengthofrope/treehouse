TreeHouse Mail System

A comprehensive email system with multiple transport drivers, queue management, template integration, file attachments, and advanced validation features.

Send Test Email

Send a simple email to test the mail system functionality.

Templated Email

Send emails using Mailable classes and TreeHouse templates.

Queue Email

Queue emails for background processing with priority support.

File Attachments

Send emails with file attachments and data attachments.

Queue Status

Monitor the mail queue status and statistics.

CLI Commands

Manage mail system via command line:

treehouse make:mailable
treehouse mail:queue:status
treehouse mail:queue:work
treehouse mail:queue:clear

Transport Drivers

  • SMTP: Production-ready with SSL/TLS support
  • Sendmail: System sendmail integration
  • Log: Development and testing driver

Advanced Features

  • Events: Mail lifecycle event handling
  • Validation: Anti-spam and security checks
  • Templates: TreeHouse template integration

Quick Start Examples

Simple Email

// Send simple email
sendMail(
    'user@example.com',
    'Welcome!',
    'Welcome to our app!'
);

Queue Email

// Queue for background processing
queueMail(
    'user@example.com',
    'Newsletter',
    'Monthly newsletter content',
    1 // High priority
);

Fluent Interface

// Fluent email building
mailer()
    ->to('user@example.com')
    ->subject('Welcome!')
    ->html('
Welcome!
')
    ->send();

Mailable Classes

// Laravel-style Mailables
$welcome = new WelcomeEmail($user);
$welcome->send($user->email);
