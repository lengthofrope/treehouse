Dot Notation Examples

Simple Variables

Page Title: {page.title}

Current Date: {page.date}

Version: {app.version}

Template Code:
<p>Page Title: {page.title}</p>

Nested Objects

User Name: {user.profile.name}

User Email: {user.profile.email}

User Role: {user.role.name}

Last Login: {user.activity.last_login}

Template Code:
<p>User Name: {user.profile.name}</p>

Configuration Values

App Name: {config.app.name}

Environment: {config.app.env}

Debug Mode: {config.app.debug}

Database Host: {config.database.host}

Template Code:
<p>App Name: {config.app.name}</p>

Dynamic Attributes (th:attr)

Dynamic Links

Visit {user.profile.name}'s Profile
Email {user.profile.name}
Template Code:
<a th:attr="href=user.profile.url">Visit Profile</a>

Dynamic Images

Profile photo of {user.profile.name}

Template Code:
<img th:attr="src=user.profile.avatar, alt=user.profile.avatarAlt">

Dynamic CSS Classes

Status: {user.status.name}

Badge: {user.badge.text}

Template Code:
<div th:attr="class=user.status.fullCssClass">

Data Attributes

Template Code:
<button th:attr="data-user-id=user.id, onclick=user.showUserOnclick">

Advanced Template Features

Conditional Rendering

Premium User

Welcome, {user.profile.name}! You have premium access.

Standard User

Hi {user.profile.name}, upgrade to premium for more features!

Template Code:
<div th:if="user.is_premium">Premium content</div>

Loops with Dot Notation

User's Recent Activities:

{act.type} {act.date}

{act.description}

Template Code:
<div th:repeat="act user.activities">{act.type}</div>
