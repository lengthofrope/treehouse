Getting Started

  • Edit your routes in config/routes/web.php
  • Create controllers in src/App/Controllers/
  • Add views in resources/views/
  • Configure your app in config/

Framework Features

  • Automatic route loading from config/routes/
  • Framework default routes at /, /api/status
  • TreeHouse template engine with th: attributes
  • Built-in cache, views, and database support
  • Support classes: Collection, Arr, Str, Carbon, Uuid

TreeHouse Template Features

  • th:text="variable" - Safe text output
  • th:html="variable" - Raw HTML output
  • th:if="condition" - Conditional rendering
  • th:repeat="item items" - Loop through arrays
  • th:class="expression" - Dynamic classes
  • {variable} - Inline expressions

Available Commands

  • ./bin/th serve - Start development server
  • ./bin/th cache:clear - Clear cache
  • ./bin/th test:run - Run tests
  • ./bin/th --help - Show all commands
