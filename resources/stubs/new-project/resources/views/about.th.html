TreeHouse Framework

TreeHouse is a modern PHP framework designed for rapid web development. Built from scratch with zero external dependencies, it provides all the essential features you need to build robust web applications.

Key Features

  • Zero Dependencies - Pure PHP implementation
  • Automatic Route Discovery - Routes load automatically from config/routes/
  • TreeHouse Template Engine - HTML-valid th: attributes
  • Built-in Cache System - File-based caching out of the box
  • Database Abstraction - ActiveRecord pattern with relations
  • Service Container - Dependency injection container
  • Console Commands - CLI tools for development
  • Support Classes - Collection, Arr, Str, Carbon, Uuid utilities

Template Syntax Examples

th:text="user.name" - Dot notation support

th:if="user.active" - Conditionals

th:repeat="item items" - Loops

th:class="user.status" - Dynamic attributes

{user.name} - Inline expressions
